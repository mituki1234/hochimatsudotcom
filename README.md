# hochimatsudotcom

**みつきのホームページ**

## 概要

このプロジェクトは、みつきさんの個人ホームページです。ブログ、欲しいものリスト、アプリ紹介などのセクションを備え、Firebaseを使用してデータを管理しています。ユーザーにとって使いやすく、魅力的なウェブサイトを提供することを目的としています。

## 使用技術

- **HTML**
- **CSS**
- **JavaScript**
- **Firebase**

## 機能

- **ブログ投稿の表示**
  - 最新のブログ記事を表示します。
- **欲しいものリスト**
  - 欲しいアイテムを一覧で紹介します。
- **アプリ紹介**
  - 開発中または公開済みのアプリへのリンクを提供します。
- **Firebase連携**
  - データの保存と取得にFirebaseを使用しています。
